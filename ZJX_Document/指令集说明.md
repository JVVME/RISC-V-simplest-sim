# RV32I Pipeline Simulator 指令子集说明

## 1. RISC-V 指令类型概览

RV32I 中，本项目涉及以下几类指令格式：

| 类型   | 用途                      |
|--------|---------------------------|
| R-type | 寄存器运算                |
| I-type | 立即数运算 / Load / JALR  |
| S-type | Store                     |
| B-type | Branch                    |
| J-type | Jump                      |

> RISC-V 所有指令固定为 **32-bit**。

---

## 2. R-type 指令

用于：`ADD` / `SUB` / `AND` / `OR` / `XOR`

### 2.1 指令格式

```
31      25 24   20 19   15 14 12 11    7 6      0
+---------+-------+-------+------+-------+--------+
| funct7  | rs2   | rs1   |funct3|  rd   | opcode |
+---------+-------+-------+------+-------+--------+
```

### 2.2 字段含义

| 字段   | 含义           |
|--------|----------------|
| opcode | 指令大类       |
| rd     | 目标寄存器     |
| rs1    | 源寄存器 1     |
| rs2    | 源寄存器 2     |
| funct3 | 子操作类型     |
| funct7 | 更细粒度控制   |

---

## 3. 算术 / 逻辑指令

### 3.1 ADD

**格式**
```
add rd, rs1, rs2
```
**行为**
```
rd = rs1 + rs2
```
**编码**

| 字段   | 值       |
|--------|----------|
| opcode | 0110011  |
| funct3 | 000      |
| funct7 | 0000000  |

## 4. I-type 指令

用于：`ADDI` / `ANDI` / `ORI` / `LW` / `JALR`

### 4.1 指令格式

```
31                20 19   15 14 12 11    7 6      0
+-------------------+-------+------+-------+--------+
|      imm[11:0]    | rs1   |funct3|  rd   | opcode |
+-------------------+-------+------+-------+--------+
```

### 4.2 字段含义

| 字段   | 含义           |
|--------|----------------|
| imm    | 12-bit 立即数  |
| rs1    | 源寄存器       |
| rd     | 目标寄存器     |
| funct3 | 子操作类型     |
| opcode | 指令类型       |

## 6. Load / Store 指令

### 6.1 LW（Load Word）

**指令类型**：I-type

**格式**
```
lw rd, imm(rs1)
```
**行为**
```
addr = rs1 + imm
rd   = Memory[addr]
```
**编码**

| 字段   | 值      |
|--------|---------|
| opcode | 0000011 |
| funct3 | 010     |

---

### 6.2 SW（Store Word）

**指令类型**：S-type

**S-type 格式**
```
31      25 24   20 19   15 14 12 11      7 6      0
+---------+-------+-------+------+----------+--------+
|imm[11:5]| rs2   | rs1   |funct3| imm[4:0] | opcode |
+---------+-------+-------+------+----------+--------+
```

**格式**
```
sw rs2, imm(rs1)
```
**行为**
```
addr        = rs1 + imm
Memory[addr] = rs2
```
**编码**

| 字段   | 值      |
|--------|---------|
| opcode | 0100011 |
| funct3 | 010     |

---

## 7. Branch 指令

用于：`BEQ` / `BNE`

### 7.1 B-type 格式

```
31 30   25 24   20 19   15 14 12 11    8 7     7 6      0
+--+-------+-------+-------+------+-------+-----+--------+
|i |imm[10:5]| rs2 | rs1   |funct3|imm[4:1]|imm11| opcode|
+--+-------+-------+-------+------+-------+-----+--------+
```

> **注意**：Branch immediate 被拆散存储。

---

### 7.2 BEQ

**格式**
```
beq rs1, rs2, offset
```
**行为**
```
if (rs1 == rs2)  PC = PC + offset
else             PC = PC + 4
```
**编码**

| 字段   | 值      |
|--------|---------|
| opcode | 1100011 |
| funct3 | 000     |

---

### 7.3 BNE

**格式**
```
bne rs1, rs2, offset
```
**行为**
```
if (rs1 != rs2)  PC = PC + offset
```
**编码**

| 字段   | 值  |
|--------|-----|
| funct3 | 001 |

---

## 8. Jump 指令

### 8.1 JAL（Jump And Link）

**J-type 格式**
```
31 30         21 20 19       12 11    7 6      0
+--+-------------+--+-----------+-------+--------+
|i | imm[10:1]   |i | imm[19:12]|  rd   | opcode |
+--+-------------+--+-----------+-------+--------+
```

**格式**
```
jal rd, offset
```
**行为**
```
rd = PC + 4
PC = PC + offset
```
**编码**

| 字段   | 值      |
|--------|---------|
| opcode | 1101111 |

---

### 8.2 JALR（Jump And Link Register）

**指令类型**：I-type

**格式**
```
jalr rd, imm(rs1)
```
**行为**
```
tmp = PC + 4
PC  = (rs1 + imm) & ~1
rd  = tmp
```

> **为什么清最低位？**
> RISC-V 规定指令地址至少 2-byte aligned，因此 `PC[0]` 必须为 0。

**编码**

| 字段   | 值      |
|--------|---------|
| opcode | 1100111 |
| funct3 | 000     |

## 9. Pipeline 中的行为

### 9.1 算术类（ADD / ADDI）

| 阶段 | 操作         |
|------|--------------|
| IF   | 取指令       |
| ID   | 读取寄存器   |
| EX   | ALU 运算     |
| MEM  | 无操作       |
| WB   | 写回寄存器   |

---

### 9.2 Load（LW）

| 阶段 | 操作                      |
|------|---------------------------|
| EX   | 计算地址：`addr = rs1 + imm` |
| MEM  | 读取 Memory               |
| WB   | 写回 `rd`                 |

---

### 9.3 Store（SW）

| 阶段 | 操作         |
|------|--------------|
| EX   | 计算地址     |
| MEM  | 写 Memory    |
| WB   | 无操作       |

---

### 9.4 Branch（BEQ / BNE）

| 阶段 | 操作                                        |
|------|---------------------------------------------|
| EX   | 比较 `rs1 == rs2`，计算 `target = PC + offset`，可能修改 PC |

> ⚠️ **Control Hazard Source**

---

### 9.5 Jump（JAL / JALR）

| 阶段 | 操作                       |
|------|----------------------------|
| EX   | 计算跳转目标，更新 PC      |
| WB   | 保存返回地址 `PC + 4`      |
